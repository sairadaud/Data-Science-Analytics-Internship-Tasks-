Task 1 – Iris Dataset Exploration
Objective

The objective of this task is to analyze the Iris dataset, understand its structure, and visualize relationships between different flower features.

Approach
Loaded dataset using Seaborn
Explored structure using shape, columns, and head
Performed statistical analysis using describe()
Visualized relationships using scatter plots, histograms, and box plots

Tools Used
Python
Pandas
Matplotlib
Seaborn

Key Insights
Petal features are more useful for classification than sepal features
Setosa species is clearly separable from others
Dataset is clean with no missing values

Outcome
Gained understanding of data exploration, visualization, and basic statistical analysis using Python.
